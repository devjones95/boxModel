Elementos de Box Model:


Padding


Padding no CSS é o espaço interno entre o conteúdo de um elemento e sua borda. Ele pode ser definido de forma uniforme para todos os lados ou individualmente para cada lado (topo, direita, baixo, esquerda). É configurado usando as propriedades padding, padding-top, padding-right, padding-bottom e padding-left. unidade pode ser pixels, porcentagem, em, entre outras. O padding afeta o tamanho total do elemento, mas não influencia o posicionamento de outros elementos ao seu redor.


Exemplos apenas usando Padding + Width e Height:

background-color: lightskyblue; width: 200px; height: 250px; padding: 20px 30px;

background-color: lightskyblue; text-align: center; width: 300px; height: 100px; padding: 90px 30px 25px 30px;

background-color: lightskyblue; text-align: center; width: 50%; height: 20%; padding: 5% 8% 3% 2%;

background-color: lightskyblue; text-align: center; width: 25%; height: 100%; padding: 150px;
