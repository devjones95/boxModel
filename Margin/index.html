Elementos de Box Model:


Margin


Margin no CSS define o espaço externo ao redor de um elemento. Pode ser ajustado para cada lado (top, right, bottom, left) individualmente ou de uma vez só.


Exemplos apenas usando Margin +Padding + Width e Height:

background-color: yellow; width: 200px; height: 250px; padding: 20px 30px; margin: 25px 30px 40px 50px;

background-color: yellow; text-align: center; width: 300px; height: 100px; padding: 90px 30px 25px 30px; margin: 100px;

background-color: yellow; text-align: center; width: 50%; height: 20%; padding: 5% 8% 3% 2%; margin: 200px 150px;

background-color: yellow; text-align: center; width: 25%; height: 100%; padding: 150px; margin: 25%;
