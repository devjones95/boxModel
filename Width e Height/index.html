Width e Height


No CSS, width e height definem as dimens√µes de um elemento. Width: Controla a largura do elemento e Height controla a altura do elemento. Ambos podem ser definidos em unidades como pixels (px), porcentagens (%), ems (em), entre outros.


Exemplos usando somente Width, Height e aplicando o text-align para centralizar o texto:

background-color: greenyellow; width: 200px; height: 250px;

background-color: greenyellow; text-align: center; width: 300px; height: 100px;

background-color: greenyellow; text-align: center; width: 50%; height: 20%;

background-color: greenyellow; text-align: center; width: 25%; height: 100%;
